功能描述
================================================================================
本样例主要展示USART外设配置为时钟同步模式时通过DMA方式收发数据。
说明：
无

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1


辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）需两个测试板，按照如下方式连接。
   测试板A:J20    测试板B:J20
      TX-------------RX
      SCK-----------SCK
      GND-----------GND
2）打开工程clk_sync_dma_tx并重新编译，下载代码测试板A。
   打开工程clk_sync_dma_rx并重新编译，下载代码测试板B。
3）先按下测试板A按键SW1，再按下测试板B按键SW1。
4）测试板A LED1(D26绿色）点亮，按SW2按键。
4）测试板B LED1(D26绿色）点亮，表示测试板B接收的数据符合预期。

================================================================================
注意
================================================================================
如使用时更改USART通道，需同时修改UART外设通道，GPIO配置、外设Clock使能和配置
NVIC功能的相应事件号。
