功能描述

本样例主要展示CMP的单一比较功能。
说明：
无

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F480-LQFP144-050-V11

辅助工具:
---------------------
示波器Tektronix DPO 2024B

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）J25通过跳冒连接，J24-VRO与J6-29连接，J8-52与示波器连接；
2）打开工程编译并全速运行；
3）调节可调电阻R138；
4）观察LED1(绿色，D26)与示波器：
   a、J8-52电平为高时，LED1点亮，表示INP电压高于INM；
   b、J8-52电平为低时，LED1熄灭，表示INP电压低于INM；

================================================================================
注意
================================================================================
更改CMP单元时，需重新配置对应的引脚、INP、INM等
