功能描述
================================================================================
本样例为芯片进入stop模式，通过EIRQ0唤醒


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）按下SW2，可观察到红灯闪烁两下，然后芯片进入stop模式（若是debug状态，能看到IDE无法连接芯片）；
4）连接PA0管脚与VCC（给PA0一个上升沿唤醒芯片）；
5）可观察到红灯闪烁（表明芯片已被唤醒）；
6）继续给PA0上升沿，芯片陷入睡眠，下一个上升沿，芯片被唤醒。
