功能描述
================================================================================
本样例展示Timera模块基本计数及同步启动功能。

说明：
本样例设置Timera为锯齿波、增计数模式，Timera单元1同步启动单元2，开启计数上溢中断，
配置计数周期为10ms，单元1每秒切换LED0状态，单元2每秒切换LED1状态，因同步启动，故
LED同时闪烁。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）观察LED0和LED1状态，同时闪烁则工作正常。
