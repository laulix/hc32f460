功能描述
================================================================================
本样例主要展示软件启动看门狗的窗口刷新和计数溢出、刷新错误造成复位的功能。

说明：
本样例设置软件启动WDT，并以约0.75秒进行一次喂狗操作，正常启动情况下LED0闪烁；按
下SW2则触发错误区间刷新动作，导致WDT触发复位动作，WDT复位后LED0常亮。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）复位按键复位芯片，LED0闪烁；
3）按下SW2触发错误刷新动作；
4）看门狗复位，LED0常亮。
