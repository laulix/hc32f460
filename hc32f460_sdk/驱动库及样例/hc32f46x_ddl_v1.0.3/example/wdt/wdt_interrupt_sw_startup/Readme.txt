功能描述
================================================================================
本样例展示WDT模块中断功能。

说明：
本样例设置软件启动WDT，并设置约1秒触发一次WDT计数溢出中断，通过SW2循环切换工作
模式（正常模式、睡眠模式），正常模式LED0闪烁，睡眠模式LED1闪烁。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）复位按键复位芯片，进入正常模式LED0闪烁；
4）按下SW2切换工作状态到睡眠模式，LED1闪烁；
5）再次按下SW2切换工作状态到正常模式，LED0闪烁；
