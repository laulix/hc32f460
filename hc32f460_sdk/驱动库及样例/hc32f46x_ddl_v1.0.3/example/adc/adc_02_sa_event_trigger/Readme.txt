功能描述
================================================================================
1）实现了 ADC1 序列 A 和 ADC2 序列 A 由事件触发扫描的用法；
2）ADC1 序列 A 和 ADC2 序列 A 扫描模式均为单次扫描；
3）ADC1 序列 A 由 Timer0 的 事件 EVT_TMR02_GCMA 每秒触发一次，ADC2 的序列 A 由外部
   引脚 PB6 下降沿触发；
4）可用串口（波特率 115200）调试助手查看 ADC1 序列 A 和 ADC2 序列 A 的触发情况。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）Micro USB 插入 J1 USBDBG 将开发板与电脑相连；
3）启动 IDE 的调试功能，断点查看程序运行情况；或全速运行用串口调试助手查看 ADC1 序列
   A 和 ADC2 序列 A 的触发情况；
4）拉低 PB6 可触发 ADC2 序列 A 扫描。
