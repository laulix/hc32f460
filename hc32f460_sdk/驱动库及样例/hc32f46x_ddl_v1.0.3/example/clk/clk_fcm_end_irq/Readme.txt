功能描述
================================================================================
本样例为时钟频率测量，用XTAL32测量PCLK1，并验证测量是否准确,。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）可观察到绿灯（LED1）亮。
