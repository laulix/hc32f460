功能描述
================================================================================
本样例为时钟频率测量，外部管脚输入参考时钟，频率异常中断。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
信号发生器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）连接信号发生器与PH2，调整信号发生器频率为256Hz以下；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行；
4）可观察到红灯（LED0）亮。
5）调节频率高于256Hz，重复步骤2）~3），可观察到绿灯亮。

================================================================================
注意
================================================================================
也可使用switch_sys_clk样例，调节output时钟频率256hz以下，作为信号发生端。
