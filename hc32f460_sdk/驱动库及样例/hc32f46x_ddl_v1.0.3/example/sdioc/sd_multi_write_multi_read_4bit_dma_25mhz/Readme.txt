功能描述
================================================================================
本样例主要展示如何通过普通速度模式25MHz，多块4bits模式，DMA方式读写SD卡。

说明：
无

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板SD卡Socket插入MicroSD卡；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行；
4）LED1（D26绿色）点亮，读写数据一致，测试结果符合预期。

================================================================================
注意
================================================================================
LED指示灯说明：
    LED0（D23红色）:表示测试结果不符合预期；
    LED1（D26绿色）:表示测试结果符合预期；
