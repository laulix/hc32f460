功能描述
================================================================================
本样例使用I2C外设通过WM8731录取和播放音频。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程
2）根据硬件情况配置，并重新编译；
   如对MIC进行录音则打开宏定义#define RECORD_MIC      1
   如对LINEIN进行录音则关闭宏定义#define RECORD_MIC      1
3）启动IDE的下载和调试功能，全速运行
4）全速运行后，短按按键SW2，LED0红灯亮，录音开始，红灯灭录音结束
5) 录音结束后，短按按键SW2，播放一次录音。
