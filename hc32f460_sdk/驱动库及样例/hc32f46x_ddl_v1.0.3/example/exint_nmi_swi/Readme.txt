功能描述
================================================================================
本样例为Ext.Interrupt/NMI Interrupt/Software Interrupt样例。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，打开菜单view-Terminal I/O，全速运行；
3）按下SW2~SW5触发外部中断，送入下降沿信号至PB11触发NMI中断
   观察LED状态以及Terminal I/O窗口的打印信息。

================================================================================
注意
================================================================================
如必要，打开line99 #define  __PRINT_TO_TERMINAL启动IAR中的Terminal I/O打印功能。
