功能描述
================================================================================
本样例为EFM模块中一次性可编程字节（otp）样例。
本样例通过main.h中const数组直接对otp区域及其锁存区域进行操作。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）打开memory窗口，可看到otp区域与main.h中预期值一致，锁存区域全为0.


================================================================================
注意
================================================================================
当使用keil工程，需在option->Debug->Settings->Flash Download里添加HC32F460 otp flash
