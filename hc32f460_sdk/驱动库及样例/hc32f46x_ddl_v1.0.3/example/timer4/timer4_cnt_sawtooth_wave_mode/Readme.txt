功能描述
================================================================================
本样例主要展示如何使用Timer4 Counter锯齿波模式。

说明：
使能Timer4计数零值匹配中断，PCLK1作为计数时钟源,主频8MHz,中断频率为100ms一次。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）LED1(D26绿色）不断闪烁，频率为5Hz，测试结果符合预期。

================================================================================
注意
================================================================================
LED指示灯说明：
    LED1（D26绿色）熄灭:表示测试结果不符合预期；
    LED1（D26绿色）闪烁:表示测试结果符合预期；
