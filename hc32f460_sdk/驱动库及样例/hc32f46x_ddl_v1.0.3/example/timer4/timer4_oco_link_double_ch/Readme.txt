功能描述
================================================================================
本样例主要展示如何使用Timer4比较输出模块OCO。

说明：
样例配置比较输出模块OCO双通道和CNT计数值匹配时，产生中断信号，翻转测试输出I/O。

使用步骤

2）编译、下载并运行；
3）观察示波器，示波器显示周期脉冲（20Hz）。
