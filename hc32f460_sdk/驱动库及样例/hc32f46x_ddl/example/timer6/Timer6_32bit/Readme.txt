功能描述

================================================================================

本样例主要展示MCU的TIMER6的级联功能，

说明：

配置timer6，hclk设置为168M，Timer61周期为0xFFFF，Timer62周期为500
Timer61计数溢出，产生OVF事件，触发Timer62向上计数加1
================================================================================

使用步骤

================================================================================
