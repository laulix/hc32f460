功能描述
================================================================================
本样例主要展示MCU的TIMER6的3组定时器同步启动（输出PWM）、同步停止、同步清零功能
说明：
配置timer6，hclk设置为168M，timer6周期为400us
Timer61：PWMA（PE09），PWMB（PW08）
Timer62：PWMA（PE11），PWMB（PW10）
Timer63：PWMA（PE13），PWMB（PW12）
