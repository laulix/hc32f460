功能描述
================================================================================
1）实现了真随机数发生器（TRNG）的中断配置和应用；
2）用延时的方式每 100 毫秒启动一次 TRNG，在中断服务函数中读取随机数，并通过
   串口（波特率 115200）将随机数打印出来。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）用 IDE 打开工程并重新编译；
2）Micro USB 数据线插入 J1 USBDBG 将开发板与电脑相连；
3）启动 IDE 调试功能，断点查看随机数值，或全速运行程序，用串口调试助手查看随机数。

================================================================================
注意
================================================================================
1）请将随机数时钟频率设置在 1MHz（含）以内，如果不能，请将其时钟频率尽可能设低。
