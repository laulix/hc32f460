功能描述
================================================================================
本样例主要展示MCU的EMB模块的PWM同相短路刹车功能（同低检测）。
说明：
EMB1用于Timer6
（EMB2-4用于Timer4）
 EMB1中断内设置软件刹车，清除同高同低刹车标志
在while（1）内做刹车处理，处理完后使能PWM输出
================================================================================

================================================================================

---------------------

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1） 示波器查看PE09和PE08
2） 下载并运行。
3） 当PWMA和PWMB同时输出低电平时，可以看到PWM输出变为高，LED0亮起
4） 延时之后PWM恢复输出
   （由于timer6的输出配置为同高同低，所以一旦PWM输出恢复会再次产生同高刹车保护并进入中断）。
