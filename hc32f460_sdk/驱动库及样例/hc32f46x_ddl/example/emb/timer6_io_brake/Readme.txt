功能描述
================================================================================
本样例主要展示MCU的EMB模块的GPIO输入刹车功能。
说明：
EMB1用于Timer6，刹车端口为PA11或PB02
（EMB2-4用于Timer4）
 EMB1中断内设置软件刹车，清除PortIn刹车标志
在while（1）内做刹车处理，处理完后使能PWM输出

================================================================================
使用步骤
================================================================================
1） 示波器查看PE09和PE08
2） PA11连接VCC
3） 下载并运行。
4） PA11连接低电平，可以看到PWM输出变为Low level，LED0亮起
5） PA11重新接Vcc，延时之后PWM恢复输出。
