功能描述
================================================================================
本样例主要展示如何使用Timer4专用比较功能――比较触发模式。

说明：
专用比较功能比较触发模式：计数器单元CNT与专用比较寄存器值匹配时，产生DCU运算
启动信号。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）LED1（D26绿色）点亮，表示专用比较功能比较触发DCU运算，DCU运算结果符合预期。

================================================================================
注意
================================================================================
LED指示灯说明：
    LED0（D23红色）:表示测试结果不符合预期；
    LED1（D26绿色）:表示测试结果符合预期；
