功能描述
================================================================================
本样例展示SPI模块3线式主从机中断传输数据功能。

说明：
本样例配置SPI为3线式，通过宏开关选择主机或从机模式，将主机代码下载到一块板子，从
机代码下载到另一块板子，通过排线将两块板子的SPI接口相连接，主机按下SW2按键，触发
一次数据收发，主机和从机对收到的数据和发送的数据进行比较，相同的LED1亮，不同则
LED0亮。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）按下SW2触发主机、从机传输数据，并在传输完成后执行比较功能；
4）比较结果相同则LED1亮，不同则LED0亮；
5）可重复按下SW2后执行3、4的操作。

================================================================================
注意
================================================================================
两块板子GND必须相连
