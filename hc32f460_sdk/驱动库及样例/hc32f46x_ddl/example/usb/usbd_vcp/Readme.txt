功能描述
================================================================================
本样例主要展示MCU的CDC VCP功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行，
2）  使用USB连接线接到PC，
3）  安装CDC VCP driver
4）  第三步成功后，windows设备管理器会出现虚拟COM口
5）  串口数据可用过EVB上的J22收发

================================================================================
注意
================================================================================
使用开发板MicroUSB接口J14
