说明：

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行，
2）  使用USB连接线接到PC，
3）  点击按键SW2、SW3、SW4、SW5，可以看到PC光标对应移动。

================================================================================
注意
================================================================================
使用开发板MicroUSB接口J14
