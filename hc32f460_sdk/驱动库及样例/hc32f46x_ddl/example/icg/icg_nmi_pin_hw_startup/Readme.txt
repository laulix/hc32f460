功能描述
================================================================================
本样例展示ICG模块中NMI功能。

说明：
关于ICG相关配置需参照mian()函数中的描述去修改hc32f46x_icg.h中的宏定义。

本样例配置硬件NMI引脚下降沿触发中断，通过PB11端口输入外部脉冲信号，每检测到一个
有效的下降沿则LED0亮灭状态切换一次。

使用步骤
================================================================================

3）PB11引脚输入脉冲信号或手动拉低，观察LED0状态切换。
