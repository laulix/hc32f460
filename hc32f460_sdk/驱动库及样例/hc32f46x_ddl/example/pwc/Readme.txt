功能描述
================================================================================
本样例为PVD1可屏蔽中断，检测电压2.8V，PVD2不可屏蔽中断检测电压2.3V。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
电压源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）使用电压源给测试板供电，并设置电压为3.3V；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行；
4）缓慢调节电压源电压至2.8V以下（2.7V）,可观察到LED0（红）灯亮；
5）继续调节电压源电压至2.3V以下（2.2V），可观察到LED0（红）灯灭，LDE2（黄）灯亮
