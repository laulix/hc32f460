---------------------
EV-HC32F460-LQFP100-050-V1.1

---------------------
无

---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）LED1（D26绿色）点亮，DCU比较中断功能测试结果符合预期。

================================================================================
LED指示灯说明：
    LED0（D23红色）:表示测试结果不符合预期；
    LED1（D26绿色）:表示测试结果符合预期；
