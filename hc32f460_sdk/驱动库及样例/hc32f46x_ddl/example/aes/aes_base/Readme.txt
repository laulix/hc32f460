功能描述
================================================================================
1）实现了 AES128 的基本配置和应用；
2）可通过串口（波特率 115200）查看加密情况。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）Micro USB 数据线插入 J1 USBDBG 将开发板与电脑相连；
2）启动 IDE 的调试功能，断点查看加解密运行情况；或全速运行程序，用串口调试助手
   查看。
