功能描述
================================================================================
本样例展示Timera模块2相正交编码相位差计数功能。

说明：
本样例设置Timera时钟源为外部端口正交编码输入，设置PE9为端口A、PE11为端口B，配置通
道B为高电平时，通道A采样到上升沿触发递加计数，配置通道B为低电平时，通道A采样到上
升沿触发递减计数，递加计数溢出则LED0切换状态，递减计数溢出则LED1切换状态。

测试环境
================================================================================

---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
函数发生器（编码器）

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）配置函数发生器，通道A、B输出频率（1000HZ）、占空比相同，将函数发生器输出通道连
接到PE9和PE11引脚；
4）修改函数发生器相位配置，B通道的相位超前A通道1/4周期，观察LED0闪烁；
5）修改函数发生器相位配置，B通道的相位滞后A通道1/4周期，观察LED1闪烁。
